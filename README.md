## 🚀 Installation & Setup

### Prerequisites
- **Blender 3.0+** (Blender 4.x recommended)
- **Python 3.8+** (included with Blender)
- **Optional**: LM Studio for production AI (sandbox mode uses Hugging Face)

### Quick Installation

1. **Download Aura V14.0**
   ```bash
   git clone https://github.com/elliotttmiller/aura.git
   cd aura
   ```

2. **Install as Blender Add-on**
   - Open Blender → Edit → Preferences → Add-ons
   - Click "Install..." and select the aura folder
   - Enable "Aura V14.0 Sentient Artisan Environment"

3. **Automatic "Aura Mode" Activation**
   - The dedicated workspace will be created automatically
   - Switch to "Aura" workspace in the top bar
   - Find the AI chat interface in the Design sidebar

4. **Start Creating** 🎨
   - Click "🚀 Activate Sentient Mode" 
   - Enter your design request in the chat
   - Watch your jewelry come to life!
