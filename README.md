# Aura V5.0 Autonomous Cognitive Architecture

## Overview
Aura V5.0 is a revolutionary jewelry design system featuring a sophisticated **Autonomous Cognitive Architecture**. Unlike traditional automated tools, Aura V5.0 operates as a fully autonomous creative architect, using a two-stage AI pipeline to generate manufacturable jewelry designs from simple text prompts with zero hardcoded creative logic.

## V5.0 Architecture - Two-Stage Autonomous AI Pipeline

### Stage 1: AI System Architect (LLM)
- **Model**: Meta-Llama-3.1-8B-Instruct via LM Studio
- **Role**: Creative director and system architect
- **Output**: Comprehensive JSON "Master Blueprint" containing all design parameters
- **Key Innovation**: Zero creative fallback logic - AI defines every aspect of the design

### Stage 2: AI Master Artisan (Shap-E) 
- **Model**: OpenAI Shap-E 3D generative model
- **Role**: 3D geometry artist
- **Input**: Rich descriptive prompt generated by LLM
- **Output**: Base 3D geometry (.obj files)

### Stage 3: Hyper-Parametric Executor (Blender)
- **Role**: Deterministic manufacturing tool
- **Function**: Flawlessly executes the AI's Master Blueprint
- **Innovation**: Contains zero creative logic - purely parameter-driven
- **Guarantee**: Same Master Blueprint = Identical 3D model every time

## Core Protocols: AURA V5.0 Directives

**Protocol 1: Absolute Cognitive Authority**  
The LLM is the single source of creative truth. All design decisions flow from the AI's Master Blueprint.

**Protocol 2: Deterministic Execution**  
The Blender processor produces identical results given identical input parameters.

**Protocol 3: Intrinsic Manufacturability**  
AI is trained to understand jewelry production constraints, ensuring all designs are manufacturable.

**Protocol 4: Empirical Validation**  
All system operations are logged and verifiable for complete transparency.

## Master Blueprint Schema

The AI generates a complete JSON Master Blueprint with this structure:

```json
{
  "creative_prompt_for_3d_model": "Rich descriptive paragraph for 3D generation",
  "shank_parameters": {
    "profile_shape": "D-Shape | Round",
    "thickness_mm": 1.5-2.5
  },
  "setting_parameters": {
    "prong_count": 4 | 6,
    "style": "Classic | Sweeping",
    "height_above_shank_mm": "Height of stone setting"
  },
  "artistic_modifier_parameters": {
    "twist_angle_degrees": 0-180,
    "organic_displacement_strength": 0.0-0.001
  }
}
```

## Service Architecture

### Three-Server Configuration
1. **AI Artist Server** (Port 8002) - Persistent Shap-E model hosting
2. **Backend Orchestrator** (Port 8001) - Two-stage AI pipeline coordination  
3. **Frontend Application** (Port 8000) - Web interface and 3D visualization

### Workflow
1. User submits creative prompt via web interface
2. Backend sends structured prompt to LM Studio for Master Blueprint generation
3. LLM creates complete design specification as JSON
4. AI server generates 3D base geometry using Shap-E model
5. Blender processor executes Master Blueprint parameters deterministically
6. Final STL file served to frontend for 3D visualization

## Installation & Setup

### Prerequisites
- Python 3.8+ with virtual environment
- LM Studio with Meta-Llama-3.1-8B-Instruct model
- Blender 4.5+ (for production use)

### Quick Start
1. **Clone the repository**:
   ```bash
   git clone https://github.com/elliotttmiller/aura.git
   cd aura
   ```

2. **Setup virtual environment**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   # or
   .\venv\Scripts\activate   # Windows
   ```

3. **Install V5.0 dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Start LM Studio** (for production):
   - Install LM Studio
   - Download Meta-Llama-3.1-8B-Instruct model
   - Start local server on port 1234

5. **Launch Aura V5.0**:
   ```bash
   python start.py
   ```

6. **Access the interface**:
   - Frontend: http://localhost:8000
   - Backend API: http://localhost:8001  
   - AI Artist: http://localhost:8002

## Usage - Autonomous Design Generation

1. **Access the Web Interface**: Navigate to http://localhost:8000
2. **Enter Creative Prompt**: Describe your vision (e.g., "twisted vine engagement ring")
3. **Specify Technical Details**: Ring size, metal type, stone specifications
4. **Generate**: Click "Generate Design" to start the autonomous AI pipeline
5. **Review Results**: Interactive 3D model appears with downloadable STL file
